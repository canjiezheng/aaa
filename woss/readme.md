# 电信采集系统实现
## 介绍  
该系统主要用于读取存放用户登录信息的文件，分析用户信息，从客户端发送至服务器，服务器处理信息，存放入数据库中。

系统主要分为6个模块：采集模块，发送模块，存储模块，备份模块，日志模块和配置模块（最重要的模块）。

## 需要知识
* 采集模块：Java集合框架应用
* 发送模块：多线程，网络编程
* 存储模块：jdbc编程,oracle
* 备份模块：JavaIO流
* 日志模块：log4j
* 配置模块：单例模式，dom4j解析xml，IOC控制反转
